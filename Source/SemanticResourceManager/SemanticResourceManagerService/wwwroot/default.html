Semantic Resource Manager Api

Central management of resources, use swagger to experiment

Specification

A resource is a set of properties for a specific concept of a user interface element (like a button or error message).

Key

To uniquely identify a resource you must provide

  • ApplicationName
  • ElementUniqueIdentifier
  • Language and Culture of the values

Values

All resources share the same set of attributes
Title
Short header
Description
A descriptive paragraph about the resource
IconUrl
A url to a small icon representing the resource
